Nearbytes

Shared spaces, local roots, cryptographic memory.

Nearbytes is a local-first shared space for files, chat, and history. The desktop app can discover common storage locations, sync updates there, and keep collaboration rooted in ordinary folders instead of opaque cloud silos.

What Nearbytes is

A space can contain files, conversations, and a durable history of change. The same space can live in multiple storage locations at once, from local disks to shared cloud folders.

If you have just installed the app on macOS, clear quarantine once so the system stops second-guessing the bundle: 

xattr -dr com.apple.quarantine "/Applications/Nearbytes.app"

This page tracks the latest desktop release directly from GitHub and recommends the right asset for your current platform.

Install flow

  1. Download the latest release for your platform.
  2. Install Nearbytes into your normal apps location.
  3. On macOS, run the `xattr` command above once after copying the app.
  4. Open Nearbytes and mount or create a space.
  5. Share ordinary folders with people, then let Nearbytes discover and use them.
